# AgenticPage

A simple, attractive static landing page deployed with **GitHub Pages**.

## 🚀 Live site
Once GitHub Pages is enabled, the site is available at:
`https://<username>.github.io/AgenticPage/`

## 📁 Structure
- `index.html` — landing page markup
- `styles.css` — styling (dark, modern, responsive)

## 🛠️ Local preview
Open `index.html` directly in a browser, or run a simple server:

```bash
python3 -m http.server 8000
# then visit http://localhost:8000
```

## 📦 Deployment
This site is served via GitHub Pages from the `main` branch (root).
